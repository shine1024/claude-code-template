# uniflow — CONTEXT

## 1. 서비스 개요
- **서비스명**: uniflow
- **목적**: 패키지로 설계된 전자결재 서비스 기능을 제공
- **주요 기능**: 결재문서 작성, 조회, 결재처리
- **주요 사용자**: 웹브라우저 또는 모바일(웹)으로 접속하는 내부 사용자
- **시스템 구성**: 웹과 모바일은 별도 시스템으로 분리 / MariaDB 통신
