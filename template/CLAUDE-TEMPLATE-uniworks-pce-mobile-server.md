# unidocu6-mobile-server — CONTEXT

## 1. 서비스 개요
- **서비스명**: unidocu6-mobile-server (UniWorks PCE Mobile 백엔드)
- **목적**: unidocu6 경비처리 솔루션의 모바일 버전 — unidocu6-core와 통합된 Java 기반 백엔드 프로젝트
- **참조 라이브러리**: `unidocu6-core` (.jar) — base class, SAP RFC 공통 구현, 공통 유틸 제공
- **분석 순서**: `unidocu6-core` → `unidocu6-mobile-server`
