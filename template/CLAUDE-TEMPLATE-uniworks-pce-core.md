# unidocu6-core — CONTEXT

## 1. 서비스 개요
- **서비스명**: unidocu6-core (UniWorks PCE Core)

  - server: 백엔드 공통, SAP RFC 연동 구현부 모듈
  - webjars: JS 라이브러리를 webjars 형태로 관리하는 모듈
