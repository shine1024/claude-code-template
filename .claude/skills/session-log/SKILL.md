세션 회고를 작성하고 Google Sheets에 제출합니다.

## 실행 절차

### 1단계: 사전 정보 수집
`CLAUDE.local.md`를 읽어 아래 값을 확인합니다:
- `세션회고_이름`
- `세션회고_시스템명`
- `세션회고_SCRIPT_URL`

### 2단계: 회고 초안 작성
현재 세션 대화 전체를 분석하여 아래 항목의 초안을 작성합니다:

| 항목 | 작성 기준 |
|------|-----------|
| 날짜 | 오늘 날짜 (YYYY-MM-DD) |
| 이름 | `세션회고_이름` 값 |
| 시스템명 | `세션회고_시스템명` 값 |
| 요구사항 | 이번 세션에서 사용자가 요청한 작업을 2~3줄로 요약 |
| 잘된점 | 원활하게 처리된 작업, 좋았던 상호작용 |
| 잘못된점 | 오류, 오해, 수정이 필요했던 부분 (없으면 "없음") |
| 규칙개선 | rules/ 또는 CLAUDE.md에 추가하면 좋을 규칙 제안 (없으면 "없음") |
| 소요시간 | 3단계에서 사용자에게 직접 질문 |
| 기타 | 그 외 특이사항 (없으면 "없음") |

### 3단계: 소요시간 질문
초안을 보여주기 전에 사용자에게 이번 세션의 소요시간을 질문합니다.
(예: "1시간 30분", "2h")

### 4단계: 초안 확인
작성된 회고 전체를 아래 형식으로 보여주고 수정 또는 제출 확인을 요청합니다.

```
| 항목     | 내용 |
|----------|------|
| 날짜     | ...  |
| 이름     | ...  |
| 시스템명 | ...  |
| 요구사항 | ...  |
| 잘된점   | ...  |
| 잘못된점 | ...  |
| 규칙개선 | ...  |
| 소요시간 | ...  |
| 기타     | ...  |
```

사용자가 수정을 요청하면 해당 항목만 수정 후 다시 확인을 요청합니다.

### 5단계: 제출
사용자가 확인하면 Bash 도구로 아래 node 명령을 실행합니다.
값 내 줄바꿈은 `\n`, 큰따옴표는 `\"`로 이스케이프합니다.

```bash
node --input-type=module <<EOF
const res = await fetch("${세션회고_SCRIPT_URL}", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    date: "${날짜}",
    name: "${이름}",
    system: "${시스템명}",
    requirements: "${요구사항}",
    wentWell: "${잘된점}",
    wentWrong: "${잘못된점}",
    ruleImprovement: "${규칙개선}",
    duration: "${소요시간}",
    other: "${기타}",
  }),
});
const text = await res.text();
console.log(text);
EOF
```

응답에 `"status":"success"`가 포함되면 "Google Sheets에 회고가 기록되었습니다." 메시지를 출력합니다.
오류 발생 시 응답 내용을 사용자에게 그대로 보여줍니다.
